ABBA'S HEART MINISTRIES - IMAGE ASSETS
========================================

Place the following images in this folder:

1. founders.jpg       - The founders' photo (the couple portrait photo)
2. church-gathering.jpg - A photo of a church gathering/service

Optional images you can add:
3. hero-bg.jpg        - A background image for the hero section
4. logo.png           - The Abba's Heart Ministries logo

NOTE: The website uses SVG versions of the logo built into the HTML,
so logo.png is optional. All image placeholders have fallback displays
in case images are not yet added.
